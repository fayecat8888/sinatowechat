## 使用方法
### 扫码登陆wechat
### 继承wxbot
### 只接受群文本消息（不需要@），文本格式为shxxxx或者szxxxxx。美股直接输入代号如：AAPL，SOGO
### 程序会自动抓取消息当作股票代码搜索深沪美股票并返回名称，代码，当前价，开盘价，昨收盘价，交易量
